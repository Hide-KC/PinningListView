# Install

Add repository to your Project's `build.gradle`.

```
allprojects {
    repositories {
        google()
        jcenter()
        
        //add
        maven {
            url "https://dl.bintray.com/hide-kc/maven"
        }
    }
}
```

Add dependencies to your app's `build.gradle`.

```
dependencies {
  implementation 'work.kcpoipoi:pinninglistview:0.0.1'
}
```
